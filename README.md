## Installation & Setup
1. Clone the repository:
   ```sh
   git clone https://github.com/AmandeepGuggi/notenest.git
   cd notenest
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```
4. Open [http://localhost:5173](http://localhost:5173) in your browser.
